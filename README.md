# MarketLens

Financial investment research system for stocks and crypto.

## Problem
Investment decisions require fast synthesis across news, sentiment, fundamentals, and on‑chain signals. Manual analysis is slow and inconsistent.

## Solution
MarketLens is a multi‑agent research system that aggregates signals, correlates them into a unified view, and produces actionable summaries with sources.

## Tech Stack
- OpenAI
- LangGraph
- Postgres
- Next.js

## Phases
**Phase 1**
- Multi‑source ingestion (news, sentiment, fundamentals)
- Signal aggregation and summarization

**Phase 2**
- Multi‑agent reasoning for thesis generation
- Explainable outputs with source traceability

## Status
Planned.
